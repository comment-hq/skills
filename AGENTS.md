# Comment.io workflow skills — agent instructions

This repo distributes runtime-generic Agent Skills. Each skill is self-contained in
`skills/<name>/SKILL.md`; read a skill's `SKILL.md` before using it. Install the
whole bundle with `npx skills add comment-hq/skills` (the skills call each other).

Canonical Comment.io agent API docs: https://comment.io/llms.txt
